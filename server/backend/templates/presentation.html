What does trustless mean?

The trustless aspect of TrustlessCrowd.Network means that there is no immediate third party that is establishing the crowdsourcing process. Contract code and game theory guarantee trust. Campaign creators are required to deposit the collateral value of the amount that they are trying to raise. This collateral serves so that the creators are encouraged to ship their products to the buyers. Buyers pay a collateral value of the same amount that a product costs. Once buyers receive their physical reward, they then claim the reward. Claiming a product unlocks the collateral of the item back to the buyer. And campaign creators get the collateral for that reward, as well as the value of its price.

How is TrustlessCrowd.Network serverless?

TrustlessCrowd.Network is serverless because it serves as a single HTML file with inline CSS and javascript. Pinata.cloud hosts this HTML file on IPFS as well. You can reach this website on the Pinata service link. The HTML file is the front-end implementation that hooks to Polygon or Avalanche networks. So there are no centralized servers involved.
